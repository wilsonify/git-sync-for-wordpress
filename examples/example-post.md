# Welcome to GitSync

This is an example Markdown file that demonstrates how to format your content for GitSync.

## Features

GitSync makes it easy to manage your WordPress content using Git:

- **Version Control**: Keep your content in Git for full version history
- **Collaboration**: Multiple authors can work together using Git workflows
- **Markdown**: Write in a simple, readable format
- **Automation**: Content automatically syncs to WordPress

## How to Use

Simply create Markdown files like this one in your Git repository, and GitSync will:

1. Pull the latest content from your repository
2. Parse the YAML frontmatter for metadata
3. Convert Markdown to HTML
4. Create or update WordPress content

## Code Example

You can include code blocks:

```php
<?php
function hello_world() {
    echo "Hello, GitSync!";
}
```

## Links and Images

You can add [links](https://wordpress.org) and images using standard Markdown syntax.
